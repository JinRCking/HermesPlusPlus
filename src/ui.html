使用方法
1. 启动代理后，设置环境变量：
  ANTHROPIC_BASE_URL=http://127.0.0.1:57421

2. 或在 Claude Code 设置中将 API 地址设为：
  http://127.0.0.1:57421/v1

3. API Key 可以随意填写，代理会自动替换为供应商的真实 Key

4. 在供应商配置中切换供应商后，模型列表自动更新
